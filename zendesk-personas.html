💡 How to Update Personas

📁 Add New Personas

  1. Upload your PDF file to the Zendesk Personas/ folder in GitHub
  2. Update the artifactsData in js/zendesk-carousel.js
  3. Add title, description, and file path
  4. The carousel will automatically include the new persona

✏️ Edit Existing Personas

  1. Replace the PDF file in the Zendesk Personas/ folder
  2. Keep the same filename to maintain automatic updates
  3. Or update the filename in js/zendesk-carousel.js
  4. Changes will reflect immediately on refresh

🤖 Copilot Automation

Future Enhancement: GitHub Copilot Agents or Loops could be configured to:

  • Automatically detect new PDFs in the folder
  • Generate metadata from PDF content
  • Update the carousel data without manual edits
  • Notify stakeholders of changes
